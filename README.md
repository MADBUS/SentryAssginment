# sentryProject-웹기반 지능형 관제 플랫폼

## 프로젝트 개요

- **수행 기간**: 25.06.05 ~ 25.07.03
- **수행 인원**: 3명
- **프로젝트 주제**: 실시간 스트리밍·AI 분석을 통한 보안 강화하고 기존 응용프로그램 기반 관제의 한계를 극복하기 위한 웹 기반 실시간 관제 솔루션

## 목적과 목표

1. **기술적 기대효과**: 웹 기반 관제 시스템 구현 → 설치 부담 없는 접근성 강화
2. **운영적 기대효과**: 관제 인력의 업무 피로도 감소
3. **사회적 기대효과**: 범죄 및 사고 사전 예방 → 시민 안전 확보 및 긴급 상황 시 신속 대응 가능
4. **브랜드 신뢰도 강화**: 안전·보안 솔루션을 제공함으로써 기업 이미지를 높이고 신뢰를 강화할 수 있음
5. **시장 차별화**: 응용프로그램에만 머물던 기존 관제와 달리 웹 기반 관제 시스템으로 차별화 가능
6. **홍보 활용**: 실시간 시연과 데모를 통해 기술력을 효과적으로 보여주고 홍보 자료로 활용 가능
7. **운영 비용 절감**: 인력 의존도를 줄여 인건비를 효율적으로 관리하고 장기적으로 비용을 낮출 수 있음
8. **리스크 비용 감소**: 사고 발생을 사전에 예방하여 보상 및 손실 비용을 크게 절약할 수 있음
9. **매출 증대 기회**: B2B, 공공기관, 보안업체 등 다양한 신규 시장에서 매출 확대 가능성이 열림

## 기술 아키텍처

- **프로그래밍 언어**: JAVA、Ｃ＃
- **프레임워크**: Spring Tool Suite, Ajax、react
- **데이터베이스**: MySQL
- **프론트엔드**: HTML, CSS, JavaScript, JQuery, Bootstrap
- **버전 관리**: Git, Github
- **배포 환경**: 로컬 서버 (Tomcat)

## 주요 기능 및 기술

- **실시간 관제 기능**: 분석 서버 ↔ 미들 서버 연동을 통한 데이터 처리
- **소통 및 협업 기능**: 관제팀 간 실시간 채팅 기능 제공
- **맵 표출 기능**: 사고 발생시 해당 화면의 지도상 위치 확인
- **검색 및 조회 기능**: 조건별 과거 영상 검색/조회
- **사용자 설정 기능**: 채팅·카메라·분석 환경 설정 지원
- **정밀 관제 기능**: 카메라 확대 기능으로 세부 영상 확인 가능
